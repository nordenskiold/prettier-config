# `@nordenskiold/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @nordenskiold/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@nordenskiold/prettier-config"
}
```

or

**Edit `.prettierrc.json`**:

```jsonc
"@nordenskiold/prettier-config"
```
